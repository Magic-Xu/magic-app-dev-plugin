# App Development Skills

Reusable Codex skills extracted from app development work.

## Skills

- `android-instrumentation-qa-guardrails`: reliable Android connected-device UI test startup and validation methodology.

## Local Install

Install a skill globally by linking or copying its folder into:

```bash
~/.codex/skills/
```

This repository is intended to be the source of truth; global installs can point back to folders in this repo.
